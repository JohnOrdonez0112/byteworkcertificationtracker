1. Introduction
The ByteWorks Certification Tracker is a web-based platform for monitoring student certifications, attempts, and outcomes. This system simplifies tracking certifications and provides a clear overview of student progress.
This manual outlines how to use the system efficiently.

2. Accessing the Certification Tracker
Website: ByteWorks Certification Tracker
No login required—just visit the website to get started.

3. Adding a Student
Go to the "Add Student" section.
Enter the Student Name and Student ID.
Click "Add to List".
The student is now saved and selectable in the dropdown for certifications and attempts.

4. Managing Certifications
4.1 Adding a New Certification
Go to "Add Certification".
Enter the Certification Name and Certification ID.
Click "Add to List".
4.2 Assigning a Certification to a Student
Certifications are linked automatically when recording attempts.
Students and certifications must already exist in the system before they can be linked through attempts.

5. Recording Certification Attempts
Navigate to "Record Certification Attempt".
Select the Student Name and Certification Name from the dropdowns.
Enter the Attempt Date.
Choose whether the result was Pass or Fail.
Click "Submit Attempt".
Each attempt is saved with:
  - Student Name
  - Certification Name
  - Attempt Date
  - Result (Pass/Fail)

6. Viewing Reports and Statistics
The "View Full Report" section provides a detailed breakdown of student and certification activity, including:
Available Reports & Data
  - Number of students that attempted certifications
  - Number of students that earned certifications (passed at least one attempt)
  - Number of attempts per certification, per student
  - List of all student names, IDs, and certifications attempted
  - List of all student names, IDs, and certifications completed
  - List of all student names, IDs, certifications attempted and completed, with date completed

To access:
Click "View Full Report".
Review the generated tables for real-time information.
Use browser tools to print or export the report if needed.

7. Support and Troubleshooting
Accidental entries: Currently, there is no delete or edit function—contact support for changes.
General Help: Email support@byteworks.com.

8. Frequently Asked Questions (FAQs)
Q: Can I track multiple attempts for the same certification and student?
A: Yes! All attempts are logged separately.

Q: What if a student passes on a later attempt?
A: The system records all attempts. The most recent "Pass" determines the earned certification.

Q: Is there a way to reset the tracker?
A: No reset function currently. Reach out to IT support for data reset.
